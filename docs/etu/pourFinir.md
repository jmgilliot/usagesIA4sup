# Dans votre établissement

Une charte, un cadre d’usage ou une modification du règlement intérieur, qu’elle soit adoptée ou en cours d’adoption reprend sans doute les principes précédents. Renseignez-vous pour connaître les règles spécifiques qui s’appliquent à votre formation. L’enseignant responsable d’un cours doit pouvoir vous renseigner sur les consignes qui s’appliquent dans son cadre. 

Il est également possible qu'un espace d'échanges soit prévu pour avancer dans les usages des IA génératives.

# Contribuez {#contribuez}

L’intelligence collective est plus forte que l’IA générative. N’hésitez pas à contribuer en proposant :

* vos usages  
* votre avis sur les usages proposés  
* des corrections  
* des illustrations  
* ou tout autre idée

Dans un premier temps, nous vous proposons pour cela : 

* Un [formulaire pour vos suggestions](https://grist.numerique.gouv.fr/o/docs/forms/uf4b75EmcxVibzWR1hemQk/69) ;
